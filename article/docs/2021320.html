IKBC C87 意见

前言

在某乎上看到的，刚好想给自己买个机械键盘，就入了，某东白色茶轴87键 349-40-30=279RMB(2021/3/19)。

包装

键盘主体x1，使用说明书x1，防尘盖x1，拔键器x1。

配置

Cherry MX 系列轴体，双色注塑 PBT，三向出线，全键无冲，两段脚撑，闭口字符，Fn键（虽然功能不如笔记本的多，但也是能用的），1.07kg。

使用感受

给的是纸盒子（环保）。
优点：1. PBT 键帽; 2. 可锁 Windows 键; 3. 有 87 键和 104 可选; 4. 有 RGB 版本。
缺点：1. 弹簧声过大; 2. 大键优化不好; 3. 轴体晃动严重; 4. 没有键线分离; 5. 无热插拔。

总评

综合来说不错，IKBC C87 在 300RMB 价位是比较优秀的，但仍有些瑕疵。如果不是 IKBC 党的话，可以考虑 GANSS GS87C，也是不错的一款机械键盘，同时因为 IKBC 换了代工厂，质量不如从前，所以可以考虑下。
