洛谷 - P5724 题解

题目描述

给出 n 和 n 个整数，求这 n 个整数中的极差是什么。极差的意思是一组数中的最大值减去最小值的差。

输入格式

无

输出格式

无

输入输出样例

Input #1

6

1 1 4 5 1 4

Output #1

4

题解

length = input()
numl = input().split()
c = 0

# 将列表里的所有元素转换成整数
while c < int(length):
    numl[c] = int(numl[c]) c +=1

if int(length) == len(numl):
    nummin = min(numl)
    nummax = max(numl)
    print(int(nummax) - int(nummin))
