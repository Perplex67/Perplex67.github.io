洛谷 - P5711 题解

题目描述

输入一个年份（大于 1582 的整数 ），判断这一年是否是闰年。

输入格式

无

输出格式

如果是闰年输出 1，否则输出 0。

输入输出样例

Input #1

1926

Output #1

0

Input #2

1900

Output #2

0

Input #3

2000

Output #3

1

Input #4

1996

Output #4

1

题解

yr = int(input())

# 判断输入的年份是否为整年
if yr % 100 == 0:
     if yr % 400 == 0:
         print(1)
     else:
         print(0)
else:
     if yr % 4 == 0:
         print(1)
     else:
         print(0)
